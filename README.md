# Grocery-Retail-Shop
Developed a complete grocery store management system that allows store owners to manage products, create customer orders, calculate totals, and view order history in an interactive dashboard. The system includes CRUD operations, dynamic UI updates, and seamless API integration with a Flask backend.

In this python project, I have build a grocery store management application. It will be 3 tier application,

Front end: UI is written in HTML/CSS/Javascript/Bootstrap
Backend: Python and Flask
Database: mysql
